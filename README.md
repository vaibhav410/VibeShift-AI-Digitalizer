## üíª Run Locally: Step-by-Step Setup

This section outlines the quick steps to set up and run the project locally on your development machine.

#### Prerequisites

* **Node.js:** (Version 18.x or newer recommended)
* **npm:** (Node Package Manager)

#### Quick Start Guide

1.  **Install Dependencies:**
    ```bash
    npm install
    ```
2.  **Set API Key:**
    Create a file named `.env.local` and set your Gemini API Key for authentication:
    ```.env
    GEMINI_API_KEY="YOUR_GEMINI_API_KEY_HERE"
    ```
3.  **Run Application:**
    ```bash
    npm run dev
    ```dev`
